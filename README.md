```bash
Usage:

    wycoin <command> [--help] [--version]
  
The commands are:
  
    node
        create <node_alias>             create a new node
        start <node_alias>              start a node
        stop <node_alias>               stop a node
        list                            list all nodes
    
    account
        create                          create a new account
        balance                         show your account address and balance
        transfer <amount> <address>     transfer an amount to another account
```
